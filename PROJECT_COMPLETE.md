## 📝 File Content Standards

### Every File Should Include:

**Header Block:**
```markdown
# [File Title]

**Last Updated:** YYYY-MM-DD  
**Version:** X.X  
**Status:** Draft/Review/Complete  

---
```

**Purpose Statement:**
```markdown
## Purpose

This file [what it does and why it exists].
```

**Main Content:**
- Clear sections with headings
- Examples where applicable
- Plain language + legal terms defined
- Cross-references to related files

**Footer:**
```markdown
---

## Related Files
- Link to relevant knowledge base
- Link to related skills
- Link to templates

## Last Review
- Reviewed by: [Name]
- Date: YYYY-MM-DD
- Next review: YYYY-MM-DD
```
